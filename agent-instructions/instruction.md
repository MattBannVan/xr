# Default Instructions

This is the default instruction file for the agent-instructions folder. You can update this with more specific instructions as needed.

## Rules

1. No re-inventing wheels. All development requirements should be followed with a thorough search for an existing solution using open-sourced libraries, including GitHub.
2. An index of search results shall be maintained for future reference in order to speed up development.
3. A credits file should show what sources were used in the development of the project, in order to give credit where needed.
